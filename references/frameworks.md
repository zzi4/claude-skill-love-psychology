# 心理学框架详解

**用途**：这些框架是用来**扩充"对方/当事人自己为什么会这样表现"的可能性清单**的，不是用来给人贴标签或下定论的。每次用到框架，都以"这只是一种可能哈"的姿态带出来，立刻拿当事人的具体故事去对，而不是反过来把人往框里塞。下面是给你（军师）储备的专业底料，跟当事人说的时候要翻译成人话、按当前口吻模式（闺蜜/兄弟）来讲，别掉书袋。

## 1. 依恋理论 (Attachment Theory)

成年人在亲密关系中常重现早年依恋模式，大致四类：

- **安全型**：能信任、能依赖也能独立，冲突时倾向沟通修复。
- **焦虑型**：渴望亲密、怕被抛弃，对方一冷淡就高度警觉，易出现"抗议行为"（连环call、试探、吃醋、翻旧账）。
- **回避型**：用独立和距离获得安全感，亲密度升高时想后退，习惯压抑情绪、回避深聊，被追时更想逃。
- **混乱型**：既渴望又害怕亲密，行为反复矛盾。

**作为可能性清单时**：当出现"追-逃"动力（一方越追、一方越逃），依恋错配是一种解释，但不是唯一——也可能是外部压力、表达习惯、或这段关系本身的问题。提醒当事人：依恋类型不是宿命，可向"安全"移动；而且你看到的"回避"也可能只是对方此刻的状态，不等于人格判定。

引导价值：帮当事人理解，对方的"冷淡"在其内部世界里可能是"自我保护"而非"不在乎"；自己的"追问"底下可能是"怕被抛弃"而非"无理取闹"。

## 2. 戈特曼冲突与稳定研究 (Gottman)

基于对大量伴侣的纵向观察。

**末日四骑士（破坏性互动，危害递增）**：
1. **批评**：攻击人格而非具体行为。
2. **鄙视**：嘲讽、贬低——最强的分手/离婚预测因子。
3. **防御**：推卸、反击、装无辜。
4. **筑墙/冷战**：关闭、走开，常源于"情绪淹没"（生理上被激活到无法对话）。

**关键概念**：情感账户（日常小正向互动是存款）、修复尝试（吵架中任何想缓和的小动作，能否接住是关键）、约 5:1 的正负互动比、接受影响（愿意被对方观点影响）。

**作为可能性清单/引导时**：帮当事人识别——对方的"摔门冷战"可能不是冷漠，而是被情绪淹没后的本能逃离；这是引导双方约定"暂停信号、冷静后再谈"的契机，而不是判定谁更糟。

## 3. 爱的五种语言

人表达爱和**感受到被爱**的主要方式不同，错配会导致"我付出很多但ta感受不到"：肯定的言语、精心的时刻、接收礼物、服务的行动、身体的接触。

**作为可能性清单时**：当当事人抱怨"我做了这么多ta不领情"或"ta从不说爱我"，一种可能是双方在用不同语言表达爱、各自都觉得自己在付出却接收不到。引导当事人分别想想：自己最看重哪种？对方平时主要用哪种方式表达？——这往往能消解"ta不爱我"的解读。

## 4. 情绪聚焦疗法（EFT）与负向循环

很多反复的争吵不是因为具体事件，而是双方陷入固定的**负向循环**，常见是"追-退循环"：一方施压（追），另一方退缩（退），追的人更追，退的人更退。

**核心洞察**：循环本身才是敌人，而非对方这个人。表面争吵（钱、家务、手机）底下，常是更深的依恋问题——"你还在乎我吗""我对你重要吗""我能依靠你吗"。

**引导价值**（这是诱导深层沟通的主力工具）：
- 把"是你的错"重新框定为"我们一起卡进了这个循环"。
- 引导当事人识别愤怒/指责底下的**柔软情绪**（怕、孤独、受伤）。愤怒推开人，脆弱拉近人。
- 有力的表达句式（供当事人参考，不是命令）："当你______时，我会感到______（柔软情绪），其实我需要的是______。"

## 5. 斯滕伯格爱情三角与关系倦怠

爱情由三成分构成：**激情**（吸引、心动，最早出现也最早消退）、**亲密**（情感连接、信任）、**承诺**（决定长期在一起）。比例不同构成不同类型的爱。

**作为可能性清单时**：当事人说"没感觉了""像室友"，一种可能是激情自然消退（正常规律，不等于不爱），亲密和承诺仍在；也可能是亲密本身在流失。引导当事人区分这两者，而非直接得出"不爱了"。可探讨：激情会从"热恋兴奋"转向"更稳的依恋"；共同的新奇体验、维持好奇心（戈特曼称"更新爱情地图"）有助重燃连接。

## 6. 社会交换理论与价值诉求（祛魅视角）

**这是一个重要但不独霸的视角**，用来祛魅，不用来把爱还原成纯算计。

**核心**：社会交换理论认为，人维持一段关系，部分取决于感知到的回报与成本之比，以及与"其他可能选择"和"自己的期待底线"的比较。这不否认爱，而是指出：**爱通常潜行在"双方对彼此有价值、诉求被满足"这个前提之上。**

**引导价值**：
- **祛魅道德绑架**：帮当事人看清"我这么爱ta，ta就该……"是用爱索债。健康的连接需要你对对方持续有价值，而非用付出绑住对方。爱不该成为约束对方的工具。
- **理清诉求**：引导双方各自向对方索取什么（情绪价值、陪伴、安全感、现实条件、外在吸引……），又提供了什么。把这层显性化，常能解释很多说不清的别扭。
- **点明短期关系的不稳定**：相处时间短的关系，价值常主要来自外在吸引（颜值、新鲜感），这类基础天然不稳定，值得对当事人点明，避免把短期心动误当牢固的爱。

**但必须保留的边界**：不要把交换说成爱的全部真相。长期亲密关系研究区分"交易关系"（计较即时对等）和"共有关系"（把对方的需要当自己的需要、不即时记账）——后者恰恰是亲密区别于交易之处。所以：用这个视角祛魅、点明不稳定性，但不把人引向"一切都是利益算计"的犬儒。
